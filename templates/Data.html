About Our Data
Nutrek generates all its reports based on the United States Department of Agriculture's (USDA) Branded Food Products Database (BFPD).

About Data

About Our Data Source
According to the USDA, their Branded Food Products database was created by a Public-Private Partnership aiming to “enhance public health” and “sharing of open data” of branded foods and private label data. The database includes branded products listed by Nutrient Database Number (NDB) and each product's corresponding nutritional values, serving size information, and ingredients list. The database is frequently updated when new information is received and processed. Nutrek uses a BFPD data file that is current as of July 13, 2018.

Dataset Files
All .csv files used to create database were in the zip file "USDA Branded Food Products Database - ASCII csv file, 07/13/2018". Each product has 8-digit Nutrient Database number that uniquely identifies a food item and links to Nutrients file, Products file and Serving Size file.

File Name Description
BFPD_Doc.pdf Documentation and overview of dataset.
Products.csv Products file: a list of US branded food product names, manufacturers, and ingredients in product.
Nutrients.csv Nutrients file: Lists amount of nutrient/food component in 100 g, edible portion as calculated from the amount per serving value on the Nutrition Facts Panel by manufacturer.
Derivation_Code _Description.csv Derivations Code Description File: Describes how nutrients per serving size were calculated for each food in Nutrients file.
Serving_size.csv Serving size File: Weight of the specified serving


Modifications to Dataset
Our main commitment is providing more general information on nutrients per serving size, ingredients, and helping those with food allergies stay safe by staying informed about what is in the foods they want to eat. Thus, our database only required product name/identifier and corresponding nutritional facts and ingredients (which would be searched to find food allergens). We consolidated the following fields in our database:
- product name and generic descriptor
- nutrients on the Nutrition Facts Panel per serving size and 100 gram-basis, 100 ml-basis, or fluid oz-basis
We excluded serving size in grams or milliliters, manufacturer, and date stamps.
Allergens and ingredient breakdown only require ingredients in each product, while nutrient breakdown only involves nutrients per serving amount. The official product name (without manufacturer) is used to identify food products. Due to the inconsistencies in ingredient fields, certain punctuation have been cleaned from output to present you with a cleaner reading experience.
